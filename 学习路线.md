## 第一阶段：HTML & CSS（大二暑假第 1-2 周）

### 1.1 HTML（第 1 周，每天 2 小时）

你不需要背 100 个标签。学完下面这些就能写页面了。

**必学标签：**

```html
<!-- 页面骨架 -->

<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>页面标题</title>
</head>
<body>
  <!-- 页面内容放这里 -->
</body>
</html>

<!-- 常用标签 -->
<h1> ~ <h6>               标题（从大到小）
<p>                       段落
<a href="网址">链接文字</a>  超链接
<img src="图片地址" alt="描述">  图片
<ul> <li>                 无序列表
<ol> <li>                 有序列表
<div>                     块级容器（最常用）
<span>                    行内容器

<form>
  <input type="text" placeholder="请输入">
  <input type="password" placeholder="请输入密码">
  <input type="email">
  <input type="number">
  <textarea></textarea>
  <select> <option> </select>
```

### 1.2 CSS（第 2 周，每天 2 小时）

CSS 东西很多，你只需要学**布局**和**美化**两部分。

**必学属性：**

```css
/* === 文字相关 === */
color: #333;              /* 文字颜色 */
font-size: 16px;          /* 字号 */
font-weight: bold;        /* 粗细 */
text-align: center;       /* 对齐 */
line-height: 1.5;         /* 行高 */

/* === 盒子模型（最重要！）=== */
width: 300px;             /* 宽 */
height: 200px;            /* 高 */
margin: 20px;             /* 外边距（元素之间的距离） */
padding: 16px;            /* 内边距（元素内部留白） */
border: 1px solid #ccc;   /* 边框 */

/* === 背景和颜色 === */
background-color: #f5f5f5;
background-image: url(...);
border-radius: 8px;       /* 圆角 */

/* === Flex 布局（核心，现代页面全靠它）=== */
display: flex;
justify-content: center;  /* 水平居中 */
align-items: center;      /* 垂直居中 */
flex-direction: column;   /* 纵向排列 */
gap: 16px;                /* 子元素间距 */
```

**学习方式：**

| 天数 | 内容 | 输出 |
|------|------|------|
| Day 1-3 | 看 [MDN CSS 入门](https://developer.mozilla.org/zh-CN/docs/Learn/CSS/First_steps) + Flex 章节 | 把每个属性都试一遍 |

| Day 6-7 | **动手：美化个人介绍页** | 把上一周的页面加上颜色、布局、间距，做得像个人主页 |

**必学概念：**

```jsx
// 1. 组件（函数组件）
function MyButton() {
  return <button>点我</button>;
}

// 2. JSX：在 JS 里写 HTML
const name = "小明";
<h1>你好，{name}</h1>

// 3. Props：父组件传数据给子组件
function Greeting({ name }) {
  return <h1>你好，{name}</h1>;
}
<Greeting name="小明" />

// 4. useState：状态管理（数据变了，页面自动更新）
const [count, setCount] = useState(0);
<button onClick={() => setCount(count + 1)}>点击 {count} 次</button>

// 5. useEffect：副作用（数据请求、定时器等）
useEffect(() => {
  fetch('/api/data').then(res => res.json()).then(setData);
}, []);  // 空数组 = 只在组件第一次加载时执行

// 6. 条件渲染
{isLoggedIn ? <Dashboard /> : <Login />}
{items.map(item => <li key={item.id}>{item.name}</li>)}
```

### 4.3 Prisma ORM（第 17 周）

数据库操作不用手写 SQL，用 ORM 更安全。

```typescript
// schema.prisma
model User {
  id    Int     @id @default(autoincrement())
  name  String
  email String  @unique
  posts Post[]
}
model Post {
  id      Int    @id @default(autoincrement())
  title   String
  content String
  userId  Int
  user    User   @relation(fields: [userId], references: [id])
}

// 使用
const users = await prisma.user.findMany({ include: { posts: true } });
const user = await prisma.user.create({
  data: { name: '小明', email: 'xm@qq.com' }
});

// 事务（保证一组操作要么全成功要么全失败）
await prisma.$transaction([
  prisma.stock.update({ ... }),
  prisma.order.create({ ... }),
]);
```

**动手：** 用 Prisma 改写留言板 API。

**参考：** 打开你 Mini Mall 的 `prisma/schema.prisma` 和 `src/app/api/`，这就是真实的全栈代码。

### 5.3 计算机网络（大三上，2 周）

**资源：**《图解 HTTP》（薄书，两天翻完）

**必会知识点：**

```
1. 从输入 URL 到页面显示，发生了什么？
   DNS 解析 → TCP 三次握手 → HTTP 请求 → 服务器处理 → HTTP 响应 → 浏览器渲染

2. HTTP 常见状态码
   200 成功  301 永久重定向  302 临时重定向
   400 请求错误  401 未认证  403 禁止访问  404 未找到
   500 服务器错误

3. GET vs POST 的区别
4. Cookie / Session / JWT Token 的区别和使用场景
5. 什么是跨域？CORS 怎么解决？
6. HTTP 缓存：强缓存 vs 协商缓存
```

### 5.4 Git 进阶（大三上，1 周）

你已会 `add` / `commit` / `push`，补这些：

```bash
git log --oneline           # 看提交历史
git diff                    # 看改了啥
git stash                   # 临时藏起当前改动
git stash pop               # 拿出来
git branch 分支名            # 创建分支
git checkout 分支名          # 切换分支
git merge 分支名             # 合并分支
git rebase                  # 整理提交历史
git reset --soft HEAD~1     # 撤销上一次commit，改动保留
```
