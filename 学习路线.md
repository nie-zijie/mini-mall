# 本科就业学习路线（211 大二 · 零基础 · 目标本科就业）

> 适用对象：211 计算机/软件工程，大二在读，现在什么都不会  
> 目标岗位：前端 / 全栈开发工程师  
> 核心语言：TypeScript / JavaScript  
> 预计周期：大二下 → 大四秋招，约 2 年

---

## 你现在的情况

```
你是 211 计算机专业大二学生。

已经学过的（学校教的）：C 语言基础、高数、线代
还没学过的：HTML/CSS/JavaScript/React/网络/数据库/算法/...

目标是本科毕业直接就业，不是考研。

问：来得及吗？
答：大二开始完全来得及。你现在开始，大三上能找实习，大四秋招能拿 Offer。
```

---

## 整条路线的逻辑

```
让你找到工作的是什么？
  1. 有项目 → 简历能过
  2. 懂原理 → 面试能聊
  3. 会算法 → 笔试能过

所以路线分三条线并行推进：
  A 线：语言 & 项目（你的核心竞争力）
  B 线：计算机基础（面试会问的理论）
  C 线：算法（笔试硬门槛）
```

## 第一阶段：学习前端基础，做出第一个页面（大二下，约 2 个月）

> 目标：零基础 → 能用 HTML/CSS/JS 做一个完整的页面  
> 现在就开始，别等了

### 1.1 HTML（第 1 周）

**只学这些标签：**

```html
<html>  <head>  <body>           → 页面结构
<h1> ~ <h6>                       → 标题
<p>                                → 段落
<a href="">                        → 链接
<img src="">                       → 图片
<div>  <span>                      → 万能容器
<ul> <ol> <li>                     → 列表
<form> <input> <button>            → 表单
```

**资源：** B 站搜"黑马程序员 HTML 入门"，看前 5 集（1.5 倍速，2 小时看完）。或者直接看 [MDN HTML 入门](https://developer.mozilla.org/zh-CN/docs/Learn/HTML/Introduction_to_HTML/Getting_started)。

**动手：** 做一个个人介绍页面，包含：你的名字、一段自我介绍、一张图、一个超链接。

### 1.2 CSS 基础（第 2 周）

**只学这些：**

```css
color / background-color            → 颜色
font-size / font-weight             → 文字
width / height / margin / padding   → 尺寸和间距
border / border-radius              → 边框
display: flex                       → 布局（重点学这个）
```

**资源：** B 站搜"尚硅谷 CSS 基础"，看前 10 集。或者 [MDN CSS 入门](https://developer.mozilla.org/zh-CN/docs/Learn/CSS/First_steps)。

**动手：** 用 CSS 把上一周的个人介绍页美化一下。

### 1.3 JavaScript 基础（第 3-4 周）

**学基础语法：**

```javascript
变量：let、const
类型：string、number、boolean、array、object
条件：if、switch
循环：for、while
函数：function、箭头函数
```

**资源：** [现代 JavaScript 教程](https://zh.javascript.info/) 第一章到第五章（必看，免费、中文、比视频高效得多）

**动手：** 不需要在浏览器里写。用 VS Code 新建 `test.js`，用 `node test.js` 运行。

```
第 3 周：变量、类型、条件、循环
第 4 周：函数、对象、数组、常用方法（map / filter / sort）
```

### 1.4 DOM 操作 + 事件（第 5 周）

**学怎么让页面"动起来"：**

```javascript
document.querySelector()   → 找到页面元素
element.textContent         → 改文字
element.style.xxx           → 改样式
element.addEventListener()  → 监听点击、输入等事件
```

**动手：** 做一个计数器。页面上有个数字，点"+"加一，点"-"减一。

### 1.5 JavaScript 核心（第 6-8 周）

**重点啃这三个难但必考的：**

| 知识点 | 要学到什么程度 |
|--------|--------------|
| **this / 闭包 / 作用域** | 能写代码演示，能画图解释 |
| **原型链** | 知道 `__proto__` 和 `prototype` 的关系 |
| **Promise / async-await** | 能处理异步请求，知道 `then` 链怎么传 |

**资源：** [现代 JavaScript 教程](https://zh.javascript.info/) 第六到第十一章。B 站辅助搜索"技术蛋老师"，他每个概念都画动画讲。

**动手：** 用 `fetch` 请求一个公开 API，把数据渲染到页面上。比如用[这个免费 JSON API](https://jsonplaceholder.typicode.com/posts)。

### 第一阶段检查

```
✓ 能做出来一个有交互的页面（表单 + 按钮 + 数据展示）
✓ 能解释什么是闭包
✓ 能画图讲 Event Loop（异步执行原理）
```

## 第二阶段：学习 React 基础（大二下末，约 1 个月）

> 目标：理解 React 核心思想，能做简单的单页应用

### 2.1 React 入门（第 1-2 周）

**学这些：**

```jsx
JSX 语法
组件（函数组件）
props（父传子）
useState（状态管理）
useEffect（副作用）
条件渲染、列表渲染
```

**资源：** [React 官方教程](https://react.dev/learn)（官方的"快速入门"部分，2 天就能看完）

**动手：** 做一个 Todo List：
- 输入框 + 添加按钮
- 展示列表
- 点击删除
- 标记完成

### 2.2 TypeScript 基础（第 3 周）

**学这些：**

```typescript
基础类型注解：string / number / boolean / array
interface 和 type
函数类型
泛型 <T>（能看懂就行）
```

**资源：** [TypeScript 官方 Handbook](https://www.typescriptlang.org/docs/handbook/)，只看基础类型和接口部分。

**动手：** 把上一周的 Todo List 用 TypeScript 重写。

### 2.3 路由 + 状态提升（第 4 周）

学 `useContext` 或轻量状态管理，让你能做一个多页面应用。

**动手：** 做一个笔记应用（至少有列表页和详情页两个页面）。

## 第三阶段：全栈入门（大二暑假，约 2 个月）

> 目标：做出第一个全栈项目，这是你简历的核心

### 3.1 Node.js 基础 + 数据库（第 1-2 周）

**学这些：**

```
Node.js 是什么、npm 怎么用
Express（最简单的后端框架）
SQL 基础：SELECT / INSERT / UPDATE / DELETE / JOIN
Prisma（ORM 工具，像写代码一样操作数据库）
```

**资源：** MDN 的 [Express 教程](https://developer.mozilla.org/zh-CN/docs/Learn/Server-side/Express_Nodejs)

**动手：** 做一个简单的留言板 API（增删查改）。

### 3.2 Next.js 入门（第 3-4 周）

**学这些：**

```
App Router（pages → 文件夹即路由）
Server Component vs Client Component
layout、page、loading、error 文件约定
```

**资源：** [Next.js 官方教程](https://nextjs.org/learn)（从 Dashboard App 开始，跟着做一遍）

### 3.3 第一个全栈项目（第 5-8 周）

**动手：做一个"校园二手交易平台"**（你也可以选别的，比如校园论坛、个人博客）

要求：
- 用户注册登录（JWT 认证）
- 发布商品 / 浏览商品列表 / 搜索
- 管理员可以删除违规商品
- 部署到 Vercel（免费）

这就是你简历上的第一个项目。Mini Mall 代码放在那，可以参考但不要复制。自己从零开始。

**第三个阶段做完，你就有了第一个全栈项目。这时候你已经比 70% 的本科生能打了。**

## 第四阶段：补计算机基础 + 数据结构（大三上，约 2 个月）

> 目标：配合学校课程，把面试会问的理论补上

### 4.1 数据结构（跟着学校课走，额外学 4 周）

学校如果教 C++ 版数据结构，你就跟着学 C++ 基础 + STL。不要深入，学到能刷题就行。

| 数据结构 | 必须掌握的 |
|---------|----------|
| 数组 | 遍历、双指针、滑动窗口 |
| 链表 | 反转、合并、环形检测 |
| 栈 / 队列 | 实现 + 经典题（括号匹配、用栈实现队列） |
| 哈希表 | 理解冲突解决、能写 Two Sum |
| 二叉树 | 遍历（前中后序、层序）、求深度 |
| 堆 | 理解大顶堆小顶堆，能写 Top K |

**资源：** [Hello 算法](https://www.hello-algo.com/)（图解、可运行、中文）

### 4.2 计算机网络（2 周）

**只看面试会考的：**

| 知识点 | 程度 |
|--------|------|
| HTTP / HTTPS | 三次握手、四次挥手、常见状态码 |
| GET vs POST | 区别、幂等性、什么时候用哪个 |
| Cookie / Session / Token | 区别、各自使用场景 |
| 跨域 | 什么是跨域、CORS 怎么解决 |
| 浏览器输入 URL 后发生了什么 | 经典面试题，必须能完整讲一遍 |

**资源：** 《图解 HTTP》（日本人写的，读起来像漫画，两天翻完）

### 4.3 Git（1 周）

你已经会用基础了。补一下：
- `git rebase` 和 `git merge` 的区别
- 解决冲突
- `git stash` 暂存当前改动

## 第五阶段：做第二个项目 + LeetCode 启动（大三寒假，约 6 周）

### 5.1 第二个项目（3 周）

这个项目要比第一个像样。

**推荐方向：** 在线聊天室 / 实时协作白板 / 日程管理工具

**必须包含：**
- 至少一种新技术（WebSocket / 图表 / 文件上传 / 第三方登录）
- 看起来像个真实产品（不是什么学习 demo）
- README 写得像产品说明书

### 5.2 LeetCode 启动（每天 1-2 小时，持续到大四）

**第一轮：按专题刷（4 周）**

跟 [代码随想录](https://programmercarl.com/) 的顺序刷：

```
数组 → 链表 → 哈希表 → 字符串 → 栈与队列 → 二叉树 → 回溯 → 贪心 → 动态规划
```

每个专题 5-10 题。目标 100 题。

**刷题铁律：**
1. 一道题不要超过 30 分钟，想不出就看答案
2. 看懂答案后关掉自己写一遍
3. 每道题写注释——面试时你得能讲出来

## 第六阶段：准备简历 + 投实习（大三下，3-5 月）

### 6.1 简历

两份项目 + 211 学历 + 一段实习经历 = 大厂的面试入场券。

现在你简历上应该有两个项目。简历就是把每个项目用三句话说清楚：
- 这是什么项目
- 用了什么技术
- 解决了什么问题 / 有什么亮点

**示例：**

> **校园二手交易平台** — Next.js + TypeScript + Prisma + SQLite  
> • 独立完成全栈 Web 应用，实现用户的注册登录、商品发布浏览、搜索筛选  
> • 使用 JWT + httpOnly Cookie 实现安全认证，Prisma 事务防止并发下单  
> • 项目部署于 Vercel，GitHub README 完整

### 6.2 投实习

| 渠道 | 说明 |
|------|------|
| **Boss 直聘** | 体量最大，小厂到中厂都有 |
| **牛客网** | 大厂内推、面经、笔试真题 |
| **学长学姐内推** | 211 的校友资源一定要用 |

**先找日常实习（门槛比暑期实习低）。** 先上车，有了实习经历后面就好走。

## 最重要的话

```
1. 别追求"看完"，追求"做出来"。
   看完 50 小时的视频 ≠ 学会了。做一个能跑的项目 = 学会了。

2. 看不懂是正常的。 
   硬着头皮往下看，几天后回来就懂了。

3. 你才大二，时间充裕。
   每天 2 小时，坚持下去。一年后你会感谢现在的自己。
```
